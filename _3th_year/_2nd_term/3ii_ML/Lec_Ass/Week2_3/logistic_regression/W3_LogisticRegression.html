def sigmoid(s): 
    return 1/(1 + np.exp(-s)) 
 
def logistic_sigmoid_regression(X, y, w_init, eta, tol = 1e-4, max_count = 10000): 
    # method to calculate model logistic regression by Stochastic Gradient Descent method 
    # eta: learning rate; tol: tolerance; max_count: maximum iterates 
    w = [w_init]     
    it = 0 
    N = X.shape[1] 
    d = X.shape[0] 
    count = 0 
    check_w_after = 20 
    # loop of stochastic gradient descent   
    while count < max_count: 
        # shuffle the order of data (for stochastic gradient descent).  
        # and  put into mix_id  
        mix_id = np.random.permutation(N) 
        for i in mix_id: 
            xi = X[:, i].reshape(d, 1) 
            yi = y[i] 
            zi = sigmoid(np.dot(w[-1].T, xi)) 
            w_new = w[-1] + eta*(yi - zi)*xi 
            count += 1 
            # stopping criteria 
            if count%check_w_after == 0:                 
                if np.linalg.norm(w_new - w[-check_w_after]) < tol: 
                    return w 
            w.append(w_new) 
    return w

Gii thiu data

Mi thuc tnh l mt yu t ri ro tim n. C c cc yu t ri ro v nhn khu hc, hnh vi v y t.

Nhn khu hc:
Gii tnh: male - nam hoc n (nomial)
Tui: age - Tui ca bnh nhn;(Lin tc - Mc d tui c ghi b ct bt thnh s nguyn nhng khi nim tui l lin tc)

Hnh vi
Ngi ht thuc hin ti: currentSmoke - bnh nhn c ht thuc hay khng (Nomial)
Cigs Per Day: s iu thuc m mt ngi ht trung bnh trong mt ngy. (c th coi l lin tc v mt ngi c th ht bao nhiu iu, thm ch na iu.)

Tin s y t/bnh
Thuc BP: BPMed - bnh nhn c dng thuc huyt p hay khng (yes/no)
t qu trc : Prevalent Stroke - trc bnh nhn c b t qu hay khng (yes/no)
Prevalent Hyp: bnh nhn c tng huyt p hay khng (yes/no)
i tho ng: Diabetes - bnh nhn c b i tho ng hay khng (Danh nh)

Tnh trng y t (hin ti)
Tot Chol: mc cholesterol ton phn (Lin tc)
Sys HA: huyt p tm thu (Lin tc)
Dia BP: huyt p tm trng (Lin tc)
BMI: Ch s khi c th (Lin tc)
Heart Rate: nhp tim (Lin tc - Trong nghin cu y hc, cc bin s nh nhp tim mc d trn thc t l ri rc, nhng vn c coi l lin tc v s lng ln cc gi tr c th c.)
Glucose: mc glucose (Lin tc)

Bin d on (u ra y)
